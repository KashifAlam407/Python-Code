Python abs()  -  The abs() function returns the absolute value of the given number int, float or comples. If the number is a complex number, abs() returns its magnitude.
Python any()  -  The any() function returns True if any element of an iterable is True. If not, it returns False.
Python all()  -  The all() function returns True if all elements in the given iterable are true. If not, it returns False.
Python ascii()  -  The ascii() method returns a string containing a printable representation of an object. It escapes the non-ASCII characters in the string using \x, \u or \U escapes.
Python bin()  -  The bin() method converts and returns the binary equivalent string of a given integer. If the parameter isn't an integer, it has to implement __index__() method to return an integer 
Python bool()  -  The bool() function converts a value to Boolean (True or False) using the standard truth testing procedure.
Python bytearray()  -  The bytearray() method returns a bytearray object which is an array of the given bytes.
